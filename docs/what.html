"Drip Selector"

Project Description

We have decided to create a web app which gives users the ability to preview the look of items of clothing on a model. The app will allow users to select pictures of pre loaded clothes and see what they will look like together, for example a user will be able to select a t-shirt and a pair of jeans and the program will generate each of these on an avatar on the screen. This will allow the user to get an accurate representation of what the clothes will look like. Users will then be able to follow a link provided by the site which takes them to a store front that has the items of clothing they are previewing for sale. We will have different sections for Pants, t-shirts and other items of clothing to make the user experience more friendly and organized.

The clothes on the website will be clearly labeled so the user is quickly able to tell the brand and name of the clothes that they are previewing. The majority of the page will be taken up by the model that the clothes will be placed on and then next to this model will be the selection of clothes that the user can place on the model. Next to the pieces of clothing will be links that take the user to a storefront that sells that piece of clothing allowing them to quickly access a vendor for this piece.

The user will also be able to do actions like generate random outfits to preview and save outfits that they like for later viewing.

The main aspects for the functionality of the site will be typical web programming strategies such as HTML, CSS and Javascript. Event listeners to check for user input on the pictures and buttons to load up the model with the clothes will be the strategy to load the pieces of clothing on the model. Loading images from pre loaded links will then be used to create the model with the appropriate pieces of clothing. For the storage of clothes and images some databasing may be necessary to implement.

User Stories

  • As a user I want to see if a t-shirt matches a pair of jeans so that I can decide if I want to buy this t-shirt.
  • As a user I want to follow a link to a storefront that sells my previewed t-shirt so that I can purchase it.
  • As a user I want to share an outfit on social media so that I can see what my friends think

Project Motivation

  • COVID making people stay indoors/unable to shop for proper sized clothing
  • Helps with online purchasing as sizes may be different in different countries etc.
  • Allows for a visual platform for users to see how their outfits will look before purchasing.
  • Can assist people wanting to expand their wardrobe/style as well as introduce them to new styles of clothing.

Given the current situation Victoria is in, many people are not able to leave their suburbs, let alone purchase clothes and visit non-essential stores. Due to these restrictions that are in place, people are unable to receive a proper shopping experience which can lead to unexpected outcomes when online shopping, whether that be incorrect sizing or simply purchasing clothes that do not fit into their current wardrobe. We also quickly found that all members of the group share a common interest for clothing and trying new styles of clothing, as well as window shopping and browsing through various retailers and stores, prior to the current restrictions that are in place. As a result we decided to create a platform that would allow for a visual representation of what different pieces of clothing would look like together when worn in the same outfit, while also assisting online purchases in ensuring that the correct size of clothing is brought. In addition to this project being used as a tool for online shopping during COVID-19, individuals that are looking to expand their wardrobe and style would be able to see how different pieces of clothing would look like with each other before purchasing.

Minimum viable features

Selecting Clothes
You can choose clothes from a dropdown list for each different category.
Validation Test
Selecting a category should create a dropdown list where you are able to choose what clothing you want.

Clothes on Avatar
The clothes will appear on an avatar (that can be adjusted).
Validation Test
Clicking on the clothes should cause the avatar to display the selected clothing.

Saved Outfits
Clothes on the avatar can be saved as outfits and chosen at a later date.
Validation Test
By clicking on a ‘save outfit’ button, it should save the current composition of clothing and be accessible at any time.

Random Outfit Generator
Outfits can be randomly generated on the avatar.
Validation Test
Clicking a button should randomize the clothing on the avatar.

Purchasing clothes from partnered clothing stores
Clients will be directed to a store where they are able to purchase the selected clothes on their avatar.
Validation Test
Clicking the links to ensure they redirect the user to the vendors pages and function as expected.

Extended viable features

Share outfits with friends
Validation Test: Clicking a ‘share’ button should convert the current outfit on the avatar into a link that can be shared with friends.

Side View
Validation Test: Clicking the right side of the avatar should show the side profile of the avatar.

Multiple avatars
Validation Test: Multiple avatars will be shown with the aim to depict as many body sizes/shapes possible.
